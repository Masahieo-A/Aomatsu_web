# shared/ — アプリ間共有部品

複数アプリで使い回す部品を置く場所。**現状はほぼ空**（将来の共通化用の受け皿）。

```
shared/
├─ components/   ← 共有UIコンポーネント（例: ポータルへ戻るヘッダー）
├─ styles/       ← 共有スタイル・デザイントークン
└─ utils/        ← 共有ユーティリティ関数
```

## 使うときの方針

- 各アプリの `package.json` は独立しているため、共有コードを導入するときは
  コピー方式にするか、ワークスペース化するかをオーナーと相談すること。
- まずは各アプリ内で完結させ、**2つ以上のアプリで重複したら** `shared/` に切り出す。
- デザイントークンの正は [docs/design-system.md](../docs/design-system.md)。
