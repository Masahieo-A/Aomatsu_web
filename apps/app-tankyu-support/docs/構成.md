# 構成メモ（開発者向け）

- **モノレポ移設元**: git@github.com:Masahieo-A/tankyu-support.git（旧単独リポジトリ。原本は `~/Projects/tankyu-support` に残置・変更禁止）
- **フレームワーク / 構成**: Next.js 14（App Router）+ TypeScript + Tailwind CSS。AIは Google Gemini API（`@google/generative-ai`, `gemini-2.5-flash`）。Markdown表示は `react-markdown` + `remark-gfm`。モノレポ編入にあたり旧 `web/` 配下のアプリ本体をこのフォルダ直下へフラット化済み（`app`: 画面・API Route, `components`: UI部品, `lib`: Gemini呼び出し/system instruction, `types`）
- **DB / データ管理**: なし（会話履歴は保存せず、`/intersection` 画面用の一時データはブラウザの `sessionStorage` のみ使用）
- **APIキー管理**: `.env.local`（Gitignore対象）に `GEMINI_API_KEY` を設定。ひな形は `.env.example`
- **デプロイ先**: Vercel想定・未確認（Root Directory: `apps/app-tankyu-support`）。詳細は [../../docs/deployment.md](../../docs/deployment.md) を参照
- **トンマナ**: [../../docs/design-system.md](../../docs/design-system.md) に準拠（ポータルへ戻るヘッダー・カラーパレット・フォントを適用済み）
- **関連ドキュメント**: docs/要件定義.md, docs/出力仕様.md
