# apps/ フォルダについて

このフォルダには英語学習支援アプリを1アプリ1フォルダで格納します。

## フォルダ命名規則

- 小文字英字とハイフンのみ使用（例：`word-checker`, `pdf-renamer`）
- アプリの機能が一目でわかる名前にする

## 各アプリが持つべきファイル

```
apps/[app-name]/
├── index.html   ← アプリ本体
└── README.md    ← アプリの説明（詳細は親READMEを参照）
```
