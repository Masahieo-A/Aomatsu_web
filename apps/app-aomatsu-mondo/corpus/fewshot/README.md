# タスク別 few-shot 集（要件4.2 ④）

UC1〜UC5の入出力例。**本人が採点済みのもののみ**を蓄積する（score 4以上がfew-shotとして採用される目安）。
1行1例のJSONL。スキーマ:

```json
{"input": "ユーザーからの依頼文", "output": "青松AI（または本人）の出力", "score": 5, "note": "採点理由・使いどころ", "added_at": "2026-07-10"}
```

- uc1.jsonl — 下書き生成（テーマ・骨子・媒体指定 → 青松文体の下書き）
- uc2.jsonl — 青松らしさ添削（草稿 → 修正案+修正理由）
- uc3.jsonl — 企画壁打ち（企画案 → 批評・懸念・発展案）
- uc4.jsonl — 過去見解の検索（トピック → 立場・論拠の要約と出典）
- uc5.jsonl — 反応シミュレーション（文章 → 読者/同僚/管理職の受け取り予測）

各ファイルの先頭行はスキーマ確認用のサンプル（score:0）。実例が入ったら削除してよい。
