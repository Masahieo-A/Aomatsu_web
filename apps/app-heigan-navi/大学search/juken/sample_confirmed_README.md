# sample_confirmed.json について

このファイルは `併願照合アプリ_要件定義.md` のスキーマ（§4 ExamMethod / SubjectRequirement / SubjectCode / Evidence）に沿って、
`juken-data/` 配下の実際の入試要項テキスト（.txt、PDFの逐語抽出）から Claude エージェントが直接読み取って作成した**サンプルデータ**です。

- 本来のパイプライン（§5 抽出パイプライン：Gemini構造化＋原文一致検証）は通していません。エージェントが手動で `.txt` を読み、逐語引用（evidence.quote）を目視で書き写しています。
- `reviewedBy: "サンプルデータ(自動生成)"` は仮の値であり、実際の教員確認（§6 教員確定ツール）を経たものではありません。
- そのため **このデータをそのまま生徒への進路指導・出願判断に使用しないでください**。面談等で使う前に、必ず教員が `evidence.pdfFile` / `evidence.page` を辿って元のPDF（`juken-data/{大学名}/...`）と突き合わせ、内容を確認・修正した上で `reviewStatus` を確定させてください。
- 名古屋学院大学は `juken-data/名古屋学院大学/2027/` `2026_実績/` 両フォルダとも空（.txt/.pdfとも未取得）のため、本サンプルには含めていません。2027年度要項が公開され次第、収集・追加してください。
