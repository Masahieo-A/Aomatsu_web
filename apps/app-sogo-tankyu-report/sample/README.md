# sample/ — デモ用サンプルデータ

このフォルダはサイトのモック・デモ確認用のサンプルデータです。

## ファイル

| ファイル | 内容 |
|----------|------|
| `schedule.json` | 42グループ分のサンプルスケジュールデータ |

## 構成

- **6クラス × 7グループ = 42グループ**
- **3時間帯**: 第1発表（9:00〜9:45）、第2発表（10:00〜10:45）、第3発表（11:00〜11:45）
- **14教室**: 101〜114教室
- **PDF URL**: `SAMPLE-xxx-report` / `SAMPLE-xxx-slides` の形式（ダミーID）
  - 実際の発表会では Google Drive の実ファイルIDに置き換わります
  - 一部グループは資料未登録状態をシミュレートするため URL が空です

## 本番運用との違い

| 項目 | サンプル | 本番 |
|------|---------|------|
| PDF URL | ダミーID（プレビュー不可） | 実際の Google Drive ID |
| グループ名 | 1年X組 Y班（仮） | 実際のグループ名 |
| テーマ・概要 | 実在しない研究内容 | 生徒が作成した内容 |

## サイトへの反映方法

`data/schedule.json` にこのファイルの内容が反映されています。  
本番データに切り替える際は `data/schedule.json` を上書きしてください。
