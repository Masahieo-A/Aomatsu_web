# ゴールデンセット（回帰テスト用サンプル英文）

プロンプトやモデルを変更したときに品質が劣化していないかを確認するための
固定テストセットを置くディレクトリです（要件定義 §10）。

- Phase 1 以降で、レベル・トピック・長さの異なるサンプル英文を 20 本以上配置します
  （非文・極端に短い文などのエッジケースを含む）。
- `scripts/golden-test.ts`（Phase 3）が全件にパイプラインを実行し、
  `report-<日付>.md` をこのディレクトリに出力します。
- prompts/ のバージョンを変更したら回帰テストを実行し、前回レポートと比較するのが運用ルールです。
